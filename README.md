# Sound Recorder

Simple sound recorder using windows form application c#. 

## Getting Started

Clone and open with Visual Studio 2019.

### Prerequisites

Visual Studio (2019 recomended).

### Installing

1) Clone project
2) Open "Build" folder 
3) Open SoundRecorder.exec
